## Install

You can install tilejson using pip

```bash
$ pip install tilejson
```

Build from source

```bash
$ git clone https://github.com/developmentseed/tilejson.git
$ cd tilejson
$ pip install -e .
```

## CLI

```bash
$ tilejson --help
Usage: tilejson [OPTIONS] URL

  tilejson cli.

Options:
  --no-geo
  --help    Show this message and exit.
```
